# Fleet Management System (Python)

This project is a fleet management prototype designed to store and manage data about users, trucks, batteries, and telemetry records. 

## Features

- Add and view **Users**
- Add and view **Trucks**
- Add and view **Batteries**
- Add and view **Telemetry Records** linked to batteries and trucks
- Automatic JSON persistence for all data
- OOP style architecture

## Requirements

- Python 3.8+
- No external libraries required

## How to Run

1. Open a terminal in the project folder

2. Run:

3. Use the menu to:
    - Add users
    - Add trucks
    - Add batteries
    - Add telemetry records
    - View all saved data

## Notes 

- JSON files are created automaticaly if they do not exist.

- All Object relationships are restored on startup.
